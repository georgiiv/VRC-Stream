You need OBS and NodeJS!
https://nodejs.org/en
https://obsproject.com/


- Run install_dependencies.bat (Only the first time)
- Set OBS to the same settings as the pictures 1, 2 and 3 (The Stream Key doesn't matter, put 1234)

- Run start.bat
- Start start streaming from OBS
- Copy one of the two links that shows up in the console that pops up and play it in VRChat


NOTES:
- The more people in the world, the higher your upload bandwith requirement will be.
